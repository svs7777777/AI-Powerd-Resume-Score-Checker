# AI-Powerd-Resume-Score-Checker
AI-Powered Resume Score Checker is a Python-based application designed to evaluate resumes against job descriptions using Natural Language Processing (NLP) techniques. This project helps candidates understand how well their resume aligns with industry requirements and provides actionable feedback to improve their chances of selection.  


Developed an AI-powered Resume Score Checker using Python and NLP techniques to evaluate resumes against job descriptions, generate matching scores, and provide actionable improvement insights.
